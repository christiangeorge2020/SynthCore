Detailed Description

Contains voice-level modifiers and component modifer pointers that are shared across all voices. A "modifier" is any variable that may be connected to a GUI control, however modifiers are not required to be connected to anything and their default values are set in the structure. A nodifier may also be used by an outer container object to modify parameters externally!*.
