Detailed Description

The WDFBesselBSF3 object implements a 3rd order Bessel BSF NOTE: designed with Elsie www.TonneSoftware.comm.

Audio I/O:

  • Processes mono input to mono output.

Control I/F:

  • none - object is hardwired
