◆ LFOMode

enum LFOMode
strong

Enumeration for selecting the LFO mode.

  • kSync: LFO restarts with each new note on
  • kOneShot: LFO runs once
  • kFreeRun: LFO begins running upon initialization, does not reset with a new note on
